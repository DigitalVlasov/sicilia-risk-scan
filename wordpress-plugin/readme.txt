Quiz interattivo per la valutazione della sicurezza sul lavoro basato su D.Lgs. 81/08 con algoritmi avanzati di scoring e sistema di lead generation.

== Description ==

**Quiz Sicurezza sul Lavoro** è un plugin WordPress professionale che permette di integrare facilmente un quiz interattivo per la valutazione della conformità alla sicurezza sul lavoro secondo il D.Lgs. 81/08 aggiornato al 2025.

= Caratteristiche Principali =

* **Quiz Intelligente**: Algoritmi avanzati di scoring e logica condizionale per domande dinamiche
* **Conformità Normativa**: Basato su D.Lgs. 81/08 con sanzioni e riferimenti normativi aggiornati
* **Lead Generation**: Sistema integrato per catturare contatti qualificati
* **Dashboard Completa**: Pannello admin con statistiche, gestione risultati e export CSV
* **Insights Dinamici**: Genera consigli personalizzati basati sulle risposte dell'utente
* **Responsive Design**: Ottimizzato per tutti i dispositivi
* **Facile Installazione**: Un semplice shortcode per inserire il quiz ovunque

= Funzionalità Tecniche =

* Scoring multilivello con moltiplicatori settoriali
* Rilevamento automatico violazioni normative
* Calcolo sanzioni amministrative e penali
* Sistema di segmentazione utenti
* Generazione messaggi WhatsApp personalizzati
* Export dati in formato CSV
* Filtraggio e ricerca risultati
* Statistiche in tempo reale

= Come Usare =

1. Installa e attiva il plugin
2. Vai su "Quiz Sicurezza" nel menu admin
3. Personalizza le impostazioni (contatti, colori, logo)
4. Inserisci lo shortcode `[quiz_sicurezza]` in qualsiasi pagina o post
5. Visualizza e gestisci i risultati dal pannello admin

== Installation ==

= Installazione Automatica =

1. Accedi al pannello admin di WordPress
2. Vai su "Plugin" → "Aggiungi nuovo"
3. Cerca "Quiz Sicurezza sul Lavoro"
4. Clicca "Installa ora" e poi "Attiva"

= Installazione Manuale =

1. Scarica il file ZIP del plugin
2. Vai su "Plugin" → "Aggiungi nuovo" → "Carica plugin"
3. Seleziona il file ZIP e clicca "Installa ora"
4. Attiva il plugin

= Configurazione Iniziale =

1. Dopo l'attivazione vai su "Quiz Sicurezza" nel menu admin
2. Clicca su "Impostazioni"
3. Configura i tuoi dati di contatto e personalizzazioni
4. Salva le impostazioni

= Inserimento Quiz =

Usa lo shortcode `[quiz_sicurezza]` per inserire il quiz in qualsiasi pagina o post:

`[quiz_sicurezza]`

Oppure con parametri personalizzati:

`[quiz_sicurezza width="100%" height="600px"]`

== Frequently Asked Questions ==

= Il quiz è conforme alla normativa attuale? =

Sì, il quiz è basato sul D.Lgs. 81/08 aggiornato al 2025 con tutte le modifiche e integrazioni più recenti.

= Posso personalizzare le domande del quiz? =

Attualmente il quiz contiene domande predefinite ottimizzate per la valutazione della sicurezza. Personalizzazioni avanzate sono disponibili su richiesta.

= I dati degli utenti sono al sicuro? =

Sì, tutti i dati sono salvati nel database WordPress del tuo sito e non vengono inviati a servizi esterni.

= Posso esportare i risultati? =

Sì, dal pannello admin puoi esportare tutti i risultati in formato CSV con filtri avanzati per data, settore e livello di rischio.

= Il quiz funziona su dispositivi mobili? =

Sì, il quiz è completamente responsive e ottimizzato per smartphone, tablet e desktop.

= Posso cambiare i colori e il logo? =

Sì, dal pannello "Impostazioni" puoi personalizzare colore primario, logo e informazioni di contatto.
